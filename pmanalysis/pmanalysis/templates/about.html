What is PM Analysis?

PM analysis is a tool that runs statistical tests on phenotype data and displays that data in easy to read graphs.

How to get started

Get yourself registered! All you need to sign up is a valid email address.

Have an account?

First things first, head on over to our login page! Once you're in, you'll be redirected to our analysis page. This page is specific to you. Make sure you come prepared with directories to upload!

Get testing!

Now you have your very own analysis page! Chose a directory to upload right from this page. Search all your uploaded directories via the search bar at the top of the list. Select a directory by clicking right on it within the list. You can choose all the files you want in experimental and control in the pop up interface. Enter a P-Value, confidence interval, select your test and let us to the rest!

Results?

The results of your test are displayed in a bar graph. If you'd like to keep these results, simply select the drop-down menu in the upper right hand corner and choose a file format!
